## myLuck Verson1.2:
    此版本使用PyQt5+python3.6编写设计。
    
   ### 实现功能：
   #### 功能1-4与version1.1一致：
    1.点击“开始”，label内开始滚动显示人名，同时“开始”按钮也由蓝色变为红色的“停止”按钮
    2.点击“停止”按钮，label停止滚动，同时“停止”按钮也由红色变为蓝色的“开始”按钮
    3.再次点击“开始”按钮，会再次随机从mingdan.xlsx内随机取一人名开始在label内滚动显示
    4.点击右上方退出按钮退出
   #### 新增加功能：
    使用“space”快捷键控制“开始/停止”按钮
